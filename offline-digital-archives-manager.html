How ODAM Works

Register in application for use ODAM. You can login to your account from any mobile devices and have access to all your volumes.

When you click on Validate button it USB devices. If devices is not in Database yet it check all the objects and puts them to the database.
Now it works for most of the smartphones and with FAT32 and NTFS file system formatted disk drives.

Search
Show PDF
Show XML

Under the file name it displays matching phrase +100 chars before and +100 chars after it so that user will actually see what he was searching for. Also it searches through the xml files with metadata. When you click on the search result it opens actual file file with PDF or XML/Text extension. The same happens for image files.

Volumes list shows ID, Serial number, Current location scans. You can sort its by date.

When clicking on one of the volumes it shows all its objects. By default, data is displayed trimmed to 200 chars. Both search and file viewer work in the way as on the screen.

When scanning is complete you get to the Inventory session screen.

When you click "Scan" button you go to Scan QR code screen.

If QR code is OK then "Not found yet" text will disappear and Top counter will change. On the right top corner there is a "Reset" button. It resets the inventory session and all volumes have to be scanned once again.
